- USE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- UPDATE
- DELETE

### 🔹 Consultas SQL

- LEFT JOIN
- RIGHT JOIN
- LIKE
- BETWEEN
- IN

### 🔹 Tipos de Dados
- VARCHAR
